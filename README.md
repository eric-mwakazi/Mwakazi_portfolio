## Run Locally

Clone the project

```bash
  git https://github.com/eric-mwakazi/Mwakazi_portfolio.git
```

Go to the project directory

```bash
  cd Mwakazi_portfolio
```

Install dependencies

```bash
  npm install
```

Start the app

```bash
  npm start
```
